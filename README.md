# CVGenius-AI

## Overview
CVGenius-AI is a C# ASP.NET Core project designed to leverage the power of Artificial Intelligence for CV/resume processing and generation. This project provides a comprehensive solution for users to create and manage their CVs and resumes

## Project Structure
The project is organized into the following main folders:
- **Controllers**: Responsible for handling incoming web requests and routing them to the appropriate services.
- **Data**: Contains the data access layer that interacts with the database.
- **Models**: Defines the data models used across the application.
- **Services**: Contains business logic and data processing functionalities.

## Technologies Used
- **ASP.NET Core**: A powerful framework for building web applications.
- **Entity Framework Core**: Used for database operations with SQL Server.
- **QuestPDF**: Implemented for generating PDF documents, enabling users to export their resumes in a professional format.

## Features
- AI-powered resume processing and generation.
- Smooth integration with SQL Server using Entity Framework Core.
- Ability to generate high-quality PDFs for resumes using QuestPDF.
